## Project Update (v2)

Significantly enhanced the stock dashboard by improving flexibility, interactivity, and adding core financial analytics.

### Key Improvements:

- Dynamic User Input:
  - Replaced hard-coded ticker names and time periods with user-driven inputs
  - Users can now analyze any stock dynamically

- Multi-Stock Comparison:
  - Enabled comparison of multiple tickers within a single visualization
  - Improves relative performance analysis

- Correlation Analysis:
  - Implemented a stock correlation heatmap
  - Helps identify relationships and diversification opportunities between assets

- Advanced Financial Metrics:
  - Added volatility calculation for risk assessment
  - Implemented weighted returns for portfolio-level analysis
  - Improved statistical insights beyond basic price tracking

### Impact:

This update transforms the project from a static stock viewer into an interactive financial analysis tool with basic portfolio analytics capabilities, making it more aligned with real-world finance use cases.
